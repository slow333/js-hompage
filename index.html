javascript function 분리 구성하기

html page에서 script 적용 시에 <script defer type="module" src="위치/파일">을 추가해야함.
그리고 package.json 파일에 

{"type": "module",
 "dependencies": ...
}
를 추가해야함

그리고 공유하고자 하는 파일에 export {함수1, 함수2, ...}
사용하는 파일에서 import { 함수1, 함수2, ...} from "./공유하는파일" 해주어야함

그리고 함수 파일을 분리해서 별도로 구성하고
원하는 값을 얻는 내용은 해당 html 파일에 넣어야함.
안 그러고 함수를 정의하고 appendChild 를 사용하면 모든 html 파일에 해당 기능이 적용되어 모든 html 파일에 appendChild가 수행됨

그러니까 function은 별도의 파일로 정의하고 사용하는 쪽에서 함수를 호출하여 사용해야 함

  display: flex;
  justify-items: center;

  justify-content: space-between;
  gap: 2rem;
  list-style-type: none;
  font-size: 1.6vw;
  font-weight: 600;
  padding: 1vh 20px 0 20px;
  text-align: center;
}
