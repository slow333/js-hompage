js에서 html dom 위치 이동하기

parent node, element로 이동 하기

parentElement는 노드의 부모 요소를 반환하는데, 부모가 없거나 부모가 DOM 요소가 아니라면 null을 반환한다. parentElement의 반환값은 언제나 DOM 요소이거나 null이다.

parentNode가 부모가 요소인지 아닌지 따지지 않고 그냥 요소 노드를 반환해 준다면 parentElement의 경우에는 부모가 꼭 요소여야만 Element 형식으로 반환하고 그렇지 않으면 null을 반환한다는 것이다.

const el = document.getElementById("greeting"); el.parentElement.parentElement; el.parentNode.parentElement;

child node, element로 이동 하기

firstChild는 first childe node를 element node 또는 text node, comment node로 반환한다.

firstElementChild는 first child node를 element node로만 반환을 한다 (text와 comment nodes를 무시).

따라서 textContent에 접근하거나 style을 바꾼다든지 등등 HTML code에 반영을 위한 접근이 필요하다면 firstElementChild을 쓴다.

const el = document.getElementById("greeting"); el.firstElementChild.textContent; el.lastElementChild.style;

sibling(형재 간) node, element로 이동 하기

sibling 간 이동하면 테그 간을 하나의 text로 인식함

const el = document.getElementById("greeting"); el.nextSibling; el.previousSibling;

node 추가/제거 하기

el.appendChild(newElement); el.parentElement.removeChild(panel)
